# 🌳 CSES Tree Algorithms Solutions

A collection of accepted **C++17** solutions for the **Tree Algorithms** section of the **CSES Problem Set**. This repository is intended to help students and competitive programmers understand standard tree algorithms and techniques used in programming contests.

---

## 📚 Problem Set

The problems are from the **CSES Problem Set – Tree Algorithms**.

🔗 https://cses.fi/problemset/list/

---

## 🚀 Topics Covered

* Tree Traversal (DFS & BFS)
* Tree Diameter
* Tree Distances
* Tree Matching
* Tree DP
* Binary Lifting
* Lowest Common Ancestor (LCA)
* Subtree Queries
* Path Queries
* Euler Tour Technique
* Rerooting DP
* Centroid Concepts
* Distance Queries
* Ancestor Queries

---

## 🛠️ Language

* **C++17**
